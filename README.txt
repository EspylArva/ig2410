This folder contains:
- The english report, as pdf file (ia-project/IG2410-AI-report.pdf)
- The jupyter notebook file, as ipynb file (ia-project/src/main.ipynb)
- The different modules used during the project development, as py files (ia-project/src/*.py)
- The resource files, especially the CSV data file (ia-project/resources/covid.csv) and the figures used in the jupyter notebook, representing values obtained at run time (ia-project/resources/*.png)

To run the project as python project, please import the folder after unzipping it. The file to run is main.py (ia-project/main.py). You can either run it in an IDE, or run it as command line.

The following command line will run the script:

python -m main.py
